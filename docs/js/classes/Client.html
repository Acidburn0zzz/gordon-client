sendCustomMessage

(
  • value
)

Defined in lib\js\src\client.js:137

Send a custom binary message to the server. The message is broadcast to all users in the sender's current room. The client has to be connected and already has joined a session/room.

Parameters:

  • value Buffer

Example:

var view = new DataView(new ArrayBuffer(1 + 2));
view.setInt8(0, 11);
view.setUint16(1, 9999);
g.sendCustomMessage(view.buffer);
